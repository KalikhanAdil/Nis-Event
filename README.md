# Nis-Event
Онлайн-платформа, объединяющая информацию обо всех олимпиадах, конкурсах, хакатонах, стартапах и других мероприятиях. Пользователи смогут быстро находить актуальные события, узнавать условия участия, сроки регистрации и призы, а также планировать свое развитие и карьерный рост.

By: Kalikhan Adil, Mamleyeva Aurika, Lezhankina Safiya, Amangeldy Damir.
